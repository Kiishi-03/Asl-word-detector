Demo ASL Classifier

This is a demonstration using a simple classifier. For a real implementation, you would need to:

  • Convert your Python model to TensorFlow.js format
  • Load the converted model using tf.loadLayersModel()
  • Process the hand landmarks the same way as your Python code

Currently, this demo recognizes basic hand positions and maps them to letters.
